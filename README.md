# Flappy Bird (Godot)

A Flappy Bird clone built in Godot as a solo summer project to practice OOP and game dev fundamentals.

## How to Play

### Option 1: Download and run (Windows)
1. Go to the [Releases](https://github.com/mAliHaziq/Flappy-Bird/blob/main/Flappy%20Bird.exe) or download `FlappyBird.exe` from the repo
2. Run the .exe file
3. Press **Space** to flap, avoid the pipes!

### Option 2: Run from source (requires Godot 4.x)
1. Clone this repo
2. Open Godot, click "Import", select the `project.godot` file
3. Press F5 to run

## Features
- Classic Flappy Bird gameplay
- Day/night background transitions based on score
- Collision-based scoring system

## What I Learned
- Godot signals (emitting/catching)
- Collision shape handling
- Physics-based movement (gravity)
- Real-time input processing
